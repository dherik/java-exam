Instructions

This exam is applicable for Junior, Middle and Senior developers. So, feel free to show your skills in the exercises accord with your level. Pay more attention in produce a code with a good quality (no bugs, using OOP correctly, test, good code coverage in test, etc). The number of exercises made is not so important.

The first exercise (Hello World) is already done in the exam. As you can see in the Hello World example, split each exercise in his own package, like the Hello World example.

You can't import another libraries into maven to resolve the exercises.

Create unit tests are not mandatory, but count a lot of points.

The build cannot be broken (with errors) when you finish the exam. The mvn clean install command needs to build the project with success as result.

https://www.reddit.com/r/cscareerquestions/comments/20ahfq/heres_a_pretty_big_list_of_programming_interview/

Questions

The first 4 (four) questions are mandatory.

  1. Print the Hello World message in console
  2. Print the number 2.16 in console, without use String
  3. Write a function that determines if a string starts with an upper-case letter A-Z
  4. Write a program that prints the numbers from 1 to 100. But for multiples of three print "Fizz" instead of the number and for the multiples of five print "Buzz". For numbers which are multiples of both three and five print "FizzBuzz".
  5. Find the only element in an array that only occurs once.
  6. Find the first non-repeated character in a String
  7. Reverse a String iteratively or recursively. If the String was "people", the expected result is "elpoep"
  8. Write fibonacci iteratively or recursively
  9. Find the common elements of 2 int arrays
  10. Find the largest int value in an int array.
  11. Check if String is a palindrome
  12. Find pairs in an integer array whose sum is equal to 10
  13. Check if a String is composed of all unique characters
  14. Since a class Customer and a class Manager has a name, you need to follow this requisites for define and return a name:
    • Name can't be null
    • Name can't be empty
    • Return the first name
    • Return the last name
    • Return the name in uppercase
    • Return the name abbreviated. If the name is John Paul Van Dyk, the expected result is: John P. V. Dyk
  15. Create a class design to represent a Car, Truck and his parts: motor, engine and tires.
